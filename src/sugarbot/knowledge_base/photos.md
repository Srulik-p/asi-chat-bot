# תמונות

## העלאת תמונת פרופיל
פרופיל (לחיצה על הכינוי בצד ימין למעלה) -> לחיצה על תמונת הפרופיל הריקה (אייקון אפור) -> לטעון קובץ -> "העלה".
**זמן אישור:** ידני ע"י מנהל האתר, עד 48 שעות.

## מחיקת תמונת פרופיל
לחיצה על התמונה (שמאל למעלה) -> לחיצה נוספת -> "מחיקה / שינוי תמונה".

## מחיקה / שינוי תמונת גלריה
תפריט (3 קווים מימין) -> "פרופיל שלי" -> "גלריה" -> "עריכה" -> "מחק" -> "שמור".

## תמונה מוגנת בסיסמא (גלריה פרטית)
אייקון תמונת פרופיל -> "גלריה" -> "הוסף תמונה" -> "טען קובץ" -> בפרטיות לבחור "מוגן בסיסמא" -> לבחור סיסמא. את הסיסמא נותנים למי שרוצים שיראה את התמונה.

## שליחת תמונה בצ׳אט
בשורה הלבנה של ההתכתבות -> אייקון המצלמה -> "העלאת תמונה" -> לבחור -> לשלוח.

## שליחת תמונה חד פעמית
אחרי בחירת התמונה - לחיצה על אייקון הספרה **"1"** במקום על "שלח". התמונה תיפתח אצל המקבל פעם אחת בלבד.

## צילום מסך - לא ניתן לדעת
לא ניתן לזהות אם משתמש אחר צילם מסך. אם זה מטריד את הלקוח - להציע לשלוח את התמונה כחד פעמית (אייקון "1").

## מחיקת תמונה ששלחתי בצ׳אט
תפריט "הודעות" -> אייקון המצלמה בשורה הלבנה -> לסמן את התמונה -> "מחק תמונה".
**שים לב:** התמונה נמחקת גם אצל המקבל.

## "התמונה עדיין לא אושרה / מתי יאשרו?"
תהליך האישור **ידני**, ע"י מנהל האתר, עד כ-48 שעות מההעלאה. אם תמונה לא אושרה - משמע אינה מתאימה לקריטריונים של האתר ונמחקת אוטומטית.

## "למה לא אישרו את התמונה?"
תמונה שלא אושרה נמחקת אוטומטית, ולכן אין לנו אפשרות לדעת על איזו תמונה מדובר. אפשר לשלוח את התמונה לוואטסאפ שלנו (054-9575217) ונבדוק.

## בעיה בהעלאת תמונה
שאלות אבחון:
- מאיזה מכשיר את/ה גולש/ת?
- מה קורה כשלוחצים על "העלאת תמונה"?
- האם נתת הרשאות גישה לתמונות במכשיר?

הסיבה הכי נפוצה: ההרשאות בטלפון לא מאופשרות לדפדפן. צריך לאשר גישה לתמונות בהגדרות המכשיר.
